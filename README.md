Project Overview

This Flutter app simulates a Smart Home Dashboard allowing users to monitor and control devices like lights, fans, AC, and cameras. It demonstrates responsive UI, stateful interactivity, device control, and dynamic addition of devices.

Key Features

Dashboard: Grid of devices with icon, name, toggle switch, and status

Device Details: Larger view, control sliders/buttons, back navigation

Add Devices: FAB opens dialog to add new devices dynamically

Interactive UI: GestureDetector/InkWell for visual feedback

Responsive Layout: Flexible, Expanded, MediaQuery for all screen sizes

Consistent Theme: Custom colors, typography, modern Material Design
